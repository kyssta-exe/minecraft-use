minecraft-use

AI agent for Minecraft — connect any LLM to any server

v0.0.1-beta.1 MIT License Node.js 18+ MC 1.8 — 1.21+ 
npm install minecraft-use
npx minecraft-use start

Features

Token Efficient

~1,000 tokens per decision. State diffing, skill library, confidence engine.

Multi-LLM

Claude, GPT-4o, Ollama, or any OpenAI-compatible API.

Game Chat

Control via !mine diamond_ore or natural language.

MCP Server

Works with Claude Desktop, Hermes, and any MCP client.

Skill Library

SQLite-backed with FTS5 search. Learns from successes.

Persistent Memory

Episodic + semantic memory across sessions.

Quick Example

import { createAgent } from 'minecraft-use';

const agent = await createAgent({
  server: { host: 'localhost', port: 25565 },
  llm: { provider: 'anthropic', model: 'claude-sonnet-4-20250514' },
});

await agent.start();
await agent.mine('diamond_ore');
await agent.go(100, 64, 200);
await agent.build('a small wooden house');
await agent.stop();
